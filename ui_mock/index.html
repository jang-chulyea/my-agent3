열역학의 법칙

열역학의 법칙은 에너지 보존과 변화 방향을 설명하는 핵심 토대입니다. 냉동사이클, 압축기, 응축기, 열교환 해석 전반에서 필수적으로 연결되며, 실제 설비의 효율과 손실을 해석하는 기준이 됩니다.

Core Concepts
직관

에너지는 사라지지 않는다

계 내부의 변화는 열과 일의 출입으로 설명되며, 설비에서 발생하는 모든 변화는 결국 에너지 수지로 환원됩니다.

원리

실제 과정은 방향성이 있다

열은 고온에서 저온으로 이동하고, 실제 시스템은 항상 비가역 손실을 수반합니다.

수식

에너지 수지와 상태량

닫힌계에서는 ΔU = Q - W, 개방계에서는 엔탈피 중심 방정식을 사용합니다. 냉동장치 해석에서는 엔탈피가 더 실용적입니다.

실무

냉동사이클 해석의 기준

압축기 동력, 응축 열량, 증발 열량, 팽창 후 상태 해석은 모두 열역학 법칙 위에서 정리됩니다.

정압 팽창에서 계가 한 일은?

문제 번호 01 · 난이도 ★★☆ · 태그: 열역학, 경계일, 정압과정
문제 해석

정압 과정에서는 압력이 일정하므로, 계가 한 일은 압력과 체적 변화량의 곱으로 정리할 수 있습니다. 즉 과정의 본질은 압력이 고정된 상태에서 얼마나 체적이 늘어났는가를 읽는 것입니다.

풀이 과정
  1. 정압 조건이므로 P = constant임을 먼저 확인합니다.
  2. 경계일 식 W = ∫P dV를 적용합니다.
  3. 압력이 일정하므로 적분 결과는 W = P(V₂ - V₁)로 단순화됩니다.
  4. 체적이 증가했다면 계가 외부에 한 일은 양(+)의 값으로 해석합니다.
정답
최종 답

정압 팽창에서 계가 한 일은 W = P(V₂ - V₁) 입니다.

이상기체가 단열 압축될 때 외부에서 계에 일이 가해진다.
이 과정에서 내부에너지와 온도는 어떻게 변화하는가?

조건:
- 열출입 없음 (Q = 0)
- 비가역 손실은 무시
- 계는 닫힌계로 가정
Hybrid 업로드 문제 LLM/규칙 기반 혼합

문제 해석

단열 압축은 열교환 없이 외부 일이 계 내부 에너지 상승으로 이어지는 전형적인 닫힌계 문제입니다. 따라서 제1법칙 관점에서 입력 일이 내부에너지 증가로 연결되는지를 우선 확인해야 합니다.

풀이 과정

  1. 단열 과정이므로 Q = 0 조건을 먼저 적용합니다.
  2. 닫힌계 제1법칙에서 ΔU = Q - W를 사용합니다.
  3. 압축은 외부가 계에 일을 하는 경우이므로, 계가 한 일 W는 음(-)의 방향으로 해석됩니다.
  4. 결과적으로 ΔU는 증가하며, 이상기체라면 내부에너지 증가와 온도 상승이 함께 나타납니다.

최종 답

정리

단열 압축에서는 내부에너지 증가, 온도 상승으로 해석하는 것이 맞습니다.
